<h1>Azure - Networking</h1>

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Resource Groups)
- Remote Desktop
- Azure Virtual Networks and Subnets
- Azure Virtual Machines (Windows and Linux [Ubuntu])
